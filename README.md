# 飞书工资计算工具

基于飞书多维表格导出数据的月度工资计算工具。

## 功能

- 上传飞书多维表格导出的 Excel 文件
- 自动识别员工信息、出勤天数、请假天数等数据
- 支持自定义底薪、加班费率等参数
- 自动计算应发工资、实发工资
- 支持导出计算结果为 Excel

## 使用方式

直接打开 `index.html` 即可使用，无需安装任何依赖。

## 部署到 GitHub Pages

1. Fork 或克隆本仓库
2. 在仓库 Settings → Pages 中选择 `main` 分支
3. 访问 `https://<用户名>.github.io/feishu-salary-calculation/`
